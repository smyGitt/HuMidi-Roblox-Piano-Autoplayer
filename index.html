What is Humidify?

Humidify processes Standard MIDI files. A Standard MIDI file contains digital note data.

The application parses this data and translates it into sequential keystrokes. It utilizes the pynput library to execute these keystrokes directly to the active operating system window manager.

Humidify requires files with the .mid extension and operates optimally with piano-specific tracks.

System Functions

Stochastic Humanization

Standard macro engines utilize rigid quantization. Quantization means executing notes at mathematically exact intervals.

Humidify introduces stochastic timing variations. Stochastic means the values have random probability distributions. The algorithm modifies the start time and the articulation duration of each note by a configurable fraction of a second.

Hand Drift Correction

When stochastic timing variations accumulate, the execution will suffer from temporal desynchronization. Desynchronization means the notes no longer align with the parsed tempo map.

The application calculates a drift decay factor. This decay factor multiplies the accumulated drift value at specific synchronization points to maintain the overall track tempo.

Dynamic Pedal Actuation

The Pedal Generator analyzes the harmonic root of the parsed notes. It automatically generates keystrokes to simulate a sustain pedal.

The algorithm checks for dissonant intervals. If it detects a minor second or a tritone interval between concurrent notes, it forces a pedal release event to prevent audio overlap.

Heuristic Fingering Assignment

The system uses a heuristic algorithm to assign notes to a left hand or a right hand. A heuristic is a problem-solving approach.

It analyzes the pitch values and the temporal proximity of the notes. This enables chord-rolling execution, where notes within a block chord execute sequentially rather than simultaneously.

Configurable Error Generation

Users can configure the engine to generate deliberate mistakes.

When the mistake probability threshold is met, the system calculates a logical mistake pitch. It prioritizes adjacent diatonic keys over chromatic keys to emulate physiological performance fatigue.

Graphical Interface & Visualizer

Humidify features a graphical user interface built with the PyQt6 framework.

It includes a real-time timeline visualizer. The visualizer maps the MIDI note data to horizontal rectangles. You can monitor the active pitches and the execution progress synchronized directly to the tempo map.
